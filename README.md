# Akunify

## Prerequisite
  * [Node.js version 14.18.2](https://formulae.brew.sh/formula/node@14)
  * [Yarn](https://classic.yarnpkg.com/en/docs/install#debian-stable)

## Usage
    git clone https://github.com/backpackrid/akunify.git
    cd akunify
    yarn install

## Environment Setup

    cp .env.example .env

Running in your local

    yarn dev
